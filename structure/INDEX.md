# Frontend/Backend Structure Documentation

This folder contains all project `.md` and `.txt` files that are related to the **frontend and backend** of the Q-Ai software.

| File | Description |
|------|-------------|

| **NAVIGATION_GUIDE.md** | App page structure, routes, and user flow |
| **DEBUG_SIGNUP.md** | Debugging signup (frontend/backend) |
| **SIGNUP_FIX.md** | Backend signup fix (Supabase/RLS) |
| **FILE_ORGANIZATION_SUMMARY.md** | Backend/frontend directory and file layout |
| **DEVELOPMENT_TODO.md** | Full-stack structure and implementation todos |
| **NETWORK_ERROR_DEBUG.md** | Fixing frontend–backend network errors |
| **TESTING_GUIDE.md** | Testing backend and frontend |
| **RETRY_LOGIC_DOCUMENTATION.md** | API client retry logic (FE/BE) |
| **NEXT_STEPS.md** | Frontend/backend status and next steps |
| **PROJECT_STATUS_AND_NEXT_STEPS.md** | Overall FE/BE implementation status |
| **QUICK_TEST_SCRIPT.md** | Quick backend API test commands |
| **SETUP.md** | Mobile frontend setup (from Mobile/) |
| **README.md** | Mobile frontend readme (from Mobile/) |

Original locations of these files are unchanged; these are copies for easy reference.
